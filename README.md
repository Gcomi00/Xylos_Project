# Xylos - UX/UI Project 

## Panoramica

Il progetto ha focus sulla realizzazione di un'interfaccia per totem informativi destinati ai turisti.
L'obiettivo è quindi quello di fornire informazioni rilevanti su punti d'interesse, itinerari e servizi locali in modo immediato e accessibile. Supporta più lingue offrendo una chiara navigazione anche per le persone che non hanno famigliarità con la tecnologia, utilizzando un design user-friendly con pochi step e un layout responsive.
